# backend-learning 仓库

这是 dwanda 的后端学习仓库，用于记录从零开始学习后端技术栈的全部过程。

## 技术栈

Java 21 + Spring Boot 3.4 + Spring Cloud + Spring Cloud Alibaba + Maven + DDD + Nacos 2.0 + MySQL 8.0 + Redis 6.x + Elasticsearch 8.x + FDS + RabbitMQ + Talos

## 学习优先级

- **P0（2-3 周）**：CRUD 开发能力 — Spring Boot 分层 + JPA + MySQL CRUD，当前短期目标
- **P1（4-6 周）**：Java/Maven 深入 + Redis + Nacos + DDD 入门
- **P2（5-7 周）**：RabbitMQ + Elasticsearch
- **P3（10-16 周）**：Spring Cloud 微服务 + DDD 深入 + FDS + Talos

详见 `learning-plan/plan.md` 和 `backend-learning/学习计划.md`。

## 目录结构

```
learning-plan/     ← 学习计划
backend-learning/  ← 前端视角简化版计划
notes/             ← 知识点总结（Markdown）
code/              ← 代码实战（按技术分目录）
teaching/          ← 教学 HTML 文档
```

## 工作流程

1. 所有代码/笔记/文档直接提交到 main 分支
2. 每次学习完成后 commit + push
3. `code/` 下的项目是学习项目，不需要完美，重点是记录学习过程

## Git 约定

- 直接操作 main 分支
- commit 后自动 push
